🏢 Название фирмы не задано ОКВЭД 43.21 — Производство электромонтажных работ

Шаг 1. Название вашей компании

Введите официальное наименование вашей фирмы (оно будет отображаться во всех документах).

Вид деятельности по ОКВЭД: 43.21 «Производство электромонтажных работ» (раздел F «Строительство») — монтаж систем видеонаблюдения, СКУД, пожарной и охранной сигнализации.

Шаг 2. 📞 Входящий звонок от заказчика

Олеся Половинкина (офис-менеджер) принимает звонок.

— Здравствуйте! Это компания .... Чем можем помочь?

Клиент: Мне нужна система видеонаблюдения для частного дома по адресу: ул. Пирогова, 116.

Шаг 3. 📋 Анкета клиента

Внесите данные заказчика в CRM (все поля обязательны).

Шаг 4. 📅 Назначение визита инженера-проектировщика

Олеся Половинкина связывается с клиентом и договаривается о времени выезда.

Шаг 5. 📨 Уведомление главного инженера

🔔 ВНИМАНИЕ! Поступила новая заявка на видеонаблюдение.

Клиент: ...

Адрес: ...

Дата визита: ...

Сообщение доставлено на рабочий стол главного инженера Эскизова В.И.

Шаг 6. 📐 Эскиз проекта системы

На объект выезжает главный инженер Эскизов В.И.: исследует объект и готовит эскиз проекта — тип системы, расстановка оборудования, кабельные трассы.

Шаг 7. 📨 Уведомление для чертёжника (AutoCAD)

🔔 Чертёжникову А.С.: Вам поступил эскиз для проекта системы ....

Объект: ... (заказчик: ...)

Содержание эскиза: ...

Задача «Чертёж проекта в AutoCAD» поставлена в работу. Срок: 1 рабочий день.

Шаг 8. 📨 Уведомление бухгалтеру-сметчику

🔔 Сметовой О.П.: Вам поступил чертёж на проект системы ....

Объект: ...

Чертёж принят от Чертёжникова А.С. Задача «Локальная смета по чертежу» поставлена в работу.

Бизнес-процесс запущен: обследование, эскиз и чертёж уже выполнены, смета — в работе. Эти этапы появятся первыми задачами в диаграмме Ганта.
