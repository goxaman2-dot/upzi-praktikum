# Практикум УПЗИ

Интерактивная оболочка практикума по дисциплине
«Управление проектами по защите информации и экономика защиты информации».

Восемь взаимосвязанных практических занятий: от паспорта проекта
системы безопасности до диаграммы Ганта и сжатия критического пути.
Типы систем: СКУД, видеонаблюдение, пожарная и охранная сигнализация,
гибридные системы.

Направления подготовки: 10.03.01, 10.04.01, специальность 10.05.01.

**Открыть курс:** https://goxaman2-dot.github.io/upzi-praktikum/

Результат каждого занятия сохраняется в JSON и служит входом
следующего занятия. Кнопка «Экспорт» — сохранить работу,
«Импорт» — продолжить с сохранённого места.
